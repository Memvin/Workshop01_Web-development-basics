# Workshop 01: Web Development Basics

This project was completed as part of Workshop 01 and demonstrates foundational web development skills using HTML and CSS.

## Overview

The project includes:

- A structured HTML5 webpage
- Custom CSS styling
- A navigation menu with internal links
- A contact form with validation
- Responsive design using media queries

## Files

- `index.html` – Main webpage structure
- `css/styles.css` – Styling and responsive layout

## How to Run

1. Clone the repository
2. Open `index.html` in a web browser

## Technologies Used

- HTML5
- CSS3
- Media Queries (Responsive Design)

## Summary

This workshop demonstrates the ability to build a simple, styled, and mobile-friendly webpage using core front-end development principles.
